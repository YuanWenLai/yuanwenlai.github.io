    # 1.在watch，compute，method中获取不到this.$refs

    # 原因：页面异步渲染，无法确保页面渲染完得到$refs

    # 解决方法：

    • 用setTimeOut()异步处理，将todo处理放到页面选然后
    • 用nextTick()，将处理放到下次渲染前完成
