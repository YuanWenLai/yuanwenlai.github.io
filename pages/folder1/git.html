# git笔记

# 1.git从远程仓库克隆dev分支到本地

  • 1、初始化一个本地仓库

    git init
    1

  • 2、与远程仓库简历连接

    git remote add origin xxxx.git
    1

  • 3、查看有没有dev分支

    git branch
    1

  • 4、如果没有dev分支

    git fetch origin dev
    1

  • 5、在本地创建分支dev并切换到该分支

    git checkout -b dev origin/dev
    1

  • 6、 dev分支上的内容都拉取到本地

    git pull origin dev
    1

  • 7、 修改代码后如何更新dev

    git add .
git commit -m "这是备注信息"
git push origin dev
    1
    2
    3

  • 8、 git远程仓库地址修改，如何更新呢？

    方式一： 直接修改

    git remote set-url origin git新地址
    1

    方式二： 先删除后添加

    git remote rm origin
git remote add origin git新地址
    1
